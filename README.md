# Living Ice Temperature

Single-page web application (in `web`) and data processing (in `processing`) for the Living Ice Temperature project.

## Usage

To run the app, get [yarn](https://yarnpkg.com/getting-started/install), then:

```sh
cd app
yarn dev
```

To build the data, get [uv](https://docs.astral.sh/uv/), then:

```sh
cd processing
scripts/update-app-data
```
